PerfilService.php : 21

Servicio para la gestión de perfiles de usuario y sus privilegios asociados.

Proporciona métodos para crear, actualizar, eliminar y consultar perfiles, así como para asignar privilegios a los mismos. Maneja transacciones de base de datos y errores de manera consistente.
